# 🎓 Student Performance Analyzer using Deep Learning

This project is a Deep Learning based Student Performance Analyzer that predicts:

- Final Marks (Regression)
- Grade (Classification)
- Risk Level (Classification)

The project is deployed using Streamlit.

---

## 📌 Features
✅ Predict Student Final Marks  
✅ Predict Grade (A/B/C/D/F)  
✅ Predict Risk Level (Low/Medium/High)  
✅ Provides suggestions for improvement  
✅ Interactive Streamlit UI  

---

## 🧠 Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- TensorFlow / Keras
- Streamlit

---

## 📂 Project Workflow
1. Dataset Creation (Synthetic Dataset)
2. Data Preprocessing (Encoding + Scaling)
3. ANN Regression Model for Marks
4. ANN Classification Model for Grade
5. ANN Classification Model for Risk Level
6. Streamlit Deployment
